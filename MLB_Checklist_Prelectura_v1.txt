NOTA ACTIVA — CHECKLIST DE PRELECTURA PRE-PARTIDO (MLB)

Versión: v1 (10/05/26)
Complemento de: MLB_Heuristicas_Mercados_v2_040526_070526.txt

Propósito:
Procedimiento de chequeo antes de empezar el análisis de un partido.
Esta checklist ordena los datos y el menú de mercados disponibles.
La nota de Heurísticas v2 ordena la decisión final de qué mercado elegir.
Las dos conviven, no se reemplazan.

Naturaleza del documento:
Mantener intacta durante 2-3 semanas de uso real. Si en la práctica aparecen
desajustes con la v2 (especialmente en tipología de vulnerabilidad y criterios
de pass), recalibrar en una v2 de esta checklist.


0. DATOS MÍNIMOS ANTES DE ANALIZAR

No empezar si faltan demasiadas piezas.

[ ] Abridores confirmados.
[ ] Lineups confirmados o probables.
[ ] Bullpen usado últimos 2-3 días.
[ ] Cuotas actuales: ML, F5, RL, total, TT y props principales.
[ ] Parque.
[ ] Clima: viento, temperatura, humedad, techo si aplica.
[ ] Mano de los abridores.
[ ] Umpire si está disponible, pero solo como ajuste, no como edge principal.
[ ] Contexto de temporada: en abril/mayo, no sobrevalorar ERA, récord ni rachas cortas.


1. PRIMERA PREGUNTA: ¿CUÁL ES EL EDGE PRINCIPAL?

Escribir una frase antes de mirar 20 mercados.

[ ] El edge está en el abridor favorito.
[ ] El edge está contra el abridor rival.
[ ] El edge está en lineup vs pitcher/repertorio.
[ ] El edge está en bullpen rival.
[ ] El edge está en parque/clima.
[ ] El edge está en precio mal ajustado.
[ ] El edge está en un bateador específico.
[ ] El edge está en un pitcher prop.
[ ] No puedo nombrar el edge principal claramente.

Si no puedo escribir el edge en una frase:
Decisión inicial: pass o live.


2. TRADUCCIÓN RÁPIDA: EDGE → MERCADO

Este es el bloque más importante.

Edge principal                                  | Mercado prioritario                       | Cuidado con
------------------------------------------------+-------------------------------------------+----------------------------------------
Abridor propio domina por K/whiff + leash       | F5, Ks, outs, rival TT under              | ML full game si bullpen es malo
Abridor propio mejor pero no dominante          | Props / ML si lineup y bullpen ayudan     | F5 side automático
Abridor rival vulnerable temprano               | F5 TT, hits allowed, bases de bateadores  | Under Ks/outs si tiene leash
Abridor rival vulnerable pero con leash         | TT full game, props de bases              | F5 TT alto
Bullpen rival malo                              | TT full game, over full game, live        | TT si abridor puede esconder bullpen
Solo una ofensiva tiene ruta clara              | TT de esa ofensiva                        | Over general
Ambos equipos tienen rutas de anotar            | Over / ambos TT                           | Side si hay mucho ruido
Favorito probable pero margen dudoso            | ML / F5 protegido                         | Run line
Underdog competitivo                            | +1.5 / ML pequeño                         | +1.5 si no convierte tráfico
Pitcher con K upside pero tráfico alto          | TT rival / F5 TT rival / outs under       | Over Ks automático
Bate top con matchup claro                      | Bases / H+R+RBI / RBI / runs              | Forzar side del equipo


3. ABRIDORES

3.1 Calidad real del abridor

[ ] K% / whiff / CSW sostienen dominio real.
[ ] BB% y comando permiten trabajar profundo.
[ ] Hard contact / barrels / flyballs son manejables.
[ ] GB/FB encaja con parque y clima.
[ ] Tiene repertorio útil contra la mano dominante del lineup rival.
[ ] No depende solo de ERA/WHIP superficial.
[ ] Forma reciente confirma o contradice métricas base.
[ ] Tiene leash real: 85-100 pitcheos.
[ ] Hay restricción, regreso de lesión, opener, bulk o bullpen game.

3.2 Tipo de vulnerabilidad

No basta con decir "pitcher vulnerable".

[ ] Vulnerable por BB/tráfico.
[ ] Vulnerable por hard contact.
[ ] Vulnerable por HR/flyballs.
[ ] Vulnerable por splits de mano.
[ ] Vulnerable por bajo K%.
[ ] Vulnerable por tercera vuelta del lineup.
[ ] Vulnerable por pitch count/leash.
[ ] Vulnerable por bullpen detrás.

Traducción:

[ ] Vulnerable por BB/tráfico → TT rival / F5 TT rival / over.
[ ] Vulnerable por contacto fuerte → hits allowed / bases / TT rival.
[ ] Vulnerable por HR/flyballs → HR props / bases / TT / over.
[ ] Vulnerable por bajo K% → hits/bases, no necesariamente Ks under.
[ ] Vulnerable por leash corto → F5 TT / TT full game / bullpen angle.
[ ] Vulnerable por tercera vuelta → live / TT full game más que F5.


4. LINEUPS

4.1 Lineup confirmado

[ ] Lineup oficial revisado.
[ ] Descansan estrellas o bates clave.
[ ] Hay platoon real, no solo "muchos derechos/izquierdos".
[ ] El lineup tiene forma colectiva contra esa mano.
[ ] El orden de bateo favorece volumen de PA.
[ ] Hay bateadores fuera de posición natural o bajas defensivas relevantes.

4.2 Parte alta

[ ] 1º bateador tiene OBP real.
[ ] 2º bateador embasa o mueve corredores.
[ ] 3º-5º tienen daño: SLG, barrels, hard contact.
[ ] Hay buena secuencia OBP + poder.
[ ] Hay velocidad arriba.
[ ] Pueden elevar pitch count.
[ ] Pueden anotar sin depender solo del HR.

4.3 Parte baja

[ ] 6º-9º no son outs automáticos.
[ ] Hay contacto suficiente.
[ ] Pueden tomar BB.
[ ] Pueden extender innings.
[ ] Pueden voltear lineup.
[ ] No cortan rallies constantemente.

4.4 Perfil del lineup vs pitcher

[ ] Paciencia vs pitcher con BB/tráfico.
[ ] Bajo chase vs pitcher de comando dudoso.
[ ] Contacto vs pitcher sin whiffs.
[ ] Poder vs pitcher de flyballs/barrels.
[ ] Velocidad vs pitcher/catcher vulnerables.
[ ] Buen historial colectivo contra esa mano.
[ ] No solo uno o dos bates sostienen todo el argumento.


5. F5 VS FULL GAME

Antes de elegir mercado, definir cuándo debería aparecer el edge.

Si el edge es temprano:

[ ] Abridor propio domina claramente.
[ ] Abridor rival tiene riesgo temprano.
[ ] Lineup elegido produce en primera vuelta.
[ ] No necesito bullpen.
[ ] El F5 evita bullpen propio peligroso.

Mercados:
[ ] F5 ML.
[ ] F5 DNB / empate no acción.
[ ] F5 -0.5.
[ ] F5 TT.
[ ] Pitcher props.

Si el edge es full game:

[ ] Bullpen rival es parte importante del edge.
[ ] Abridor rival puede sobrevivir 5 innings.
[ ] Lineup elegido suele producir tarde.
[ ] Favorito tiene mejor profundidad.
[ ] Bullpen propio sostiene ventaja.
[ ] El daño puede llegar 6ª-9ª.

Mercados:
[ ] ML full game.
[ ] TT full game.
[ ] Over full game.
[ ] Run line solo si hay ruta de margen.
[ ] Props de bateadores con volumen full game.


6. BULLPEN Y PLAN DE PITCHEO

6.1 Bullpen propio

[ ] Cerrador disponible.
[ ] Setup disponible.
[ ] Brazos clave usados ayer o días consecutivos.
[ ] Bullpen descansado.
[ ] Bullpen tiene K/whiff para cerrar.
[ ] Bullpen permite HR/tráfico.
[ ] Hay long relief si abridor falla.

6.2 Bullpen rival

[ ] Bullpen gastado últimos 2-3 días.
[ ] Cerrador/setup no disponibles o cansados.
[ ] Relevistas medios vulnerables.
[ ] Bullpen permite hard contact.
[ ] Bullpen permite BB.
[ ] Hay varios zurdos/derechos para neutralizar matchups.

6.3 Opener / spot starter / bullpen game

[ ] ¿Hay bulk plan real detrás?
[ ] ¿El bulk puede cubrir 3-5 innings?
[ ] ¿El bullpen puede absorber 6-8 innings?
[ ] ¿El rival tiene paciencia/contacto para exponer el plan?
[ ] ¿El daño esperado es temprano o tarde?

Regla:
[ ] Opener/spot starter no es fade automático.
[ ] Bullpen game no significa daño temprano.
[ ] Si el edge es contra bullpen medio, preferir full game/live antes que F5.


7. PARQUE, CLIMA Y RUN ENVIRONMENT

[ ] Parque favorece HR o los reduce.
[ ] Parque favorece dobles/triples o foul territory.
[ ] Viento ayuda o frena carry.
[ ] Temperatura ayuda o reduce vuelo de pelota.
[ ] Humedad/densidad del aire relevante.
[ ] Techo abierto/cerrado.
[ ] Parque comprime rallies.
[ ] Parque aumenta varianza.

Traducción:

[ ] Parque/clima pro-carreras → TT, over, bases, HR sprinkle.
[ ] Parque pitcher-friendly → under, F5 under, +1.5 dog, pitcher props.
[ ] Coors/entornos extremos → cuidado con ML; mirar TT, +1.5, live.
[ ] Clima neutral → no forzar total solo por narrativa.


8. MONEYLINE

Buena opción si:

[ ] Edge no depende de un solo inning.
[ ] Equipo tiene mejor abridor o bullpen suficiente.
[ ] Lineup tiene ruta clara de anotación.
[ ] Bullpen no contradice el full game.
[ ] Cuota no está demasiado exprimida.
[ ] Rival no tiene ruta clara para romper el guion.

Evitar ML si:

[ ] El edge es solo del abridor y bullpen propio es malo.
[ ] El favorito gana probable pero precio es muy corto.
[ ] Lineup favorito no tiene edge ofensivo colectivo.
[ ] Rival tiene bullpen superior y dog +1.5 es más lógico.
[ ] El partido proyecta baja anotación y mucho margen de varianza.


9. F5 ML / F5 HÁNDICAP

Buena opción si:

[ ] Edge principal está en el abridor propio.
[ ] Abridor propio tiene K/whiff o comando real.
[ ] Abridor propio tiene leash para 5 innings.
[ ] Lineup propio puede anotar temprano.
[ ] Quiero evitar bullpen propio.
[ ] Rival no tiene perfil de paciencia/tráfico para elevar pitch count.

Evitar F5 si:

[ ] Abridor propio es "mejor" pero no domina por whiffs.
[ ] Rival tiene lineup paciente/contact-heavy.
[ ] Rival puede fabricar carreras sin HR.
[ ] Abridor propio suele sufrir tercera/cuarta entrada.
[ ] Lineup propio produce más tarde.
[ ] El edge real está en bullpen rival.


10. RUN LINE / HÁNDICAP

10.1 Favorito -1.5

Buena opción si:

[ ] Favorito tiene ruta real de margen.
[ ] Lineup puede hacer 5+ carreras.
[ ] Rival tiene abridor vulnerable + bullpen malo.
[ ] Bullpen favorito puede cerrar sin drama.
[ ] Rival tiene poca capacidad de backdoor.
[ ] Precio ML está muy bajo y el margen sí está justificado.

Evitar -1.5 si:

[ ] Partido proyecta cerrado.
[ ] Parque comprime carreras.
[ ] Favorito tiene edge de abridor, pero no ofensivo.
[ ] Rival tiene bullpen competente.
[ ] Favorito puede ganar 3-2 / 4-3.

10.2 Underdog +1.5

Buena opción si:

[ ] Dog tiene ruta real de anotar 3-4 carreras.
[ ] Dog convierte tráfico, no solo se embasa.
[ ] Dog tiene poder distribuido o buenos bates RBI.
[ ] Abridor dog puede evitar blow-up.
[ ] Bullpen dog no es desastre.
[ ] Favorito no tiene profundidad 1-9 para abrir margen.

Evitar +1.5 si:

[ ] Dog deja muchos corredores.
[ ] Dog tiene parte baja débil.
[ ] Abridor dog es volátil en tercera vuelta.
[ ] Favorito tiene daño distribuido.
[ ] Spot starter/opener del favorito tiene bulk plan real detrás.


11. TOTAL DEL PARTIDO

11.1 Over

Buena opción si:

[ ] Ambos abridores permiten tráfico/contacto.
[ ] Ambos lineups tienen rutas reales de anotar.
[ ] Bullpens están gastados o son vulnerables.
[ ] Parque/clima ayuda.
[ ] Hay paciencia + poder en ambos lados.
[ ] No depende de un solo equipo.

Evitar over si:

[ ] Solo una ofensiva tiene ruta clara.
[ ] Un abridor puede esconder bullpen vulnerable.
[ ] Parque comprime rallies.
[ ] Lineups incompletos.
[ ] El total ya está inflado.

11.2 Under

Buena opción si:

[ ] Ambos abridores tienen ruta a 5-6 innings sólidos.
[ ] Lineups no tienen edge colectivo real.
[ ] Parque/clima reduce carry.
[ ] Bullpens están frescos.
[ ] Favorito tiene edge de abridor, pero no ofensivo.
[ ] El partido proyecta pocos rallies, no solo mala suerte con RISP.

Evitar under si:

[ ] Hay bullpens muy usados.
[ ] Ambos lineups tienen paciencia/tráfico.
[ ] Hay parque/clima pro-carreras.
[ ] Uno o ambos abridores tienen BB alto.
[ ] Hay varios HR profiles claros.


12. TEAM TOTALS

12.1 TT over

Buena opción si:

[ ] Lineup tiene ruta clara contra abridor rival.
[ ] Pitcher rival tiene vulnerabilidad explotable por ese lineup.
[ ] Parte alta embasa.
[ ] Parte media tiene daño.
[ ] Parte baja no corta innings.
[ ] Bullpen rival también es vulnerable.
[ ] Línea no exige explosión perfecta.

Evitar TT over si:

[ ] Abridor rival puede llegar a 6-7 innings.
[ ] Lineup elegido no está completo.
[ ] TT ya está alto.
[ ] Parque/clima no ayuda.
[ ] El equipo gana probable, pero no necesariamente explota.
[ ] El edge real era ML, no carreras.

12.2 F5 TT over

Buena opción si:

[ ] Abridor rival es vulnerable temprano.
[ ] Lineup produce en primera vuelta.
[ ] Pitcher rival tiene BB/pitch count/blow-up risk.
[ ] No necesito bullpen rival para cobrar.

Evitar F5 TT over si:

[ ] Abridor rival puede sobrevivir 5 innings.
[ ] El daño proyectado es más contra bullpen.
[ ] Lineup genera tráfico, pero no convierte rápido.
[ ] Línea exige 3+ carreras tempranas sin margen.


13. PITCHER PROPS

13.1 Ks over

Buena opción si:

[ ] Pitcher tiene K/whiff/CSW reales.
[ ] Rival tiene K% alto.
[ ] Pitcher tiene leash suficiente.
[ ] No hay riesgo fuerte de salida corta.
[ ] Umpire/zona no perjudica demasiado.
[ ] Repertorio calza con debilidades del lineup.

Evitar Ks over si:

[ ] Pitcher tiene WHIP/BB/tráfico alto.
[ ] Rival tiene paciencia y OBP.
[ ] Rival puede alargar innings.
[ ] Pitcher puede salir antes de acumular Ks.
[ ] Necesita eficiencia perfecta.

13.2 Ks under

Buena opción si:

[ ] Rival tiene bajo K%.
[ ] Rival hace contacto.
[ ] Pitcher no tiene whiff real.
[ ] Leash limitado.
[ ] Pitch count puede subir por BB/tráfico.

Evitar Ks under si:

[ ] Pitcher tiene leash fuerte.
[ ] Rival tiene K naturales.
[ ] Pitcher puede permitir daño y aun así ponchar.
[ ] Línea es baja.

13.3 Outs over

Buena opción si:

[ ] Pitcher tiene leash real.
[ ] Comando estable.
[ ] Rival no eleva conteos.
[ ] Bullpen necesita descanso.
[ ] Manager suele dejarlo trabajar.

Evitar outs over si:

[ ] Pitcher viene de lesión/restricción.
[ ] Rival tiene paciencia.
[ ] Hay BB/tráfico.
[ ] Clima/parque aumenta daño.
[ ] Bullpen está descansado y puede entrar temprano.

13.4 Hits / ER allowed over

Buena opción si:

[ ] Pitcher permite contacto fuerte.
[ ] Rival tiene contacto y poder.
[ ] Pitcher tiene leash, por lo que habrá oportunidades.
[ ] Under Ks/outs no captura bien la vulnerabilidad.


14. PROPS DE BATEADORES

Las props salen del guion, no del menú.

Antes de elegir prop:

[ ] ¿El partido proyecta carreras, tráfico, poder o baja anotación?
[ ] ¿El equipo tiene TT alto o bajo?
[ ] ¿El bateador está 1º-5º?
[ ] ¿Tiene volumen probable de PA?
[ ] ¿Tiene buen matchup de mano/repertorio?
[ ] ¿Tiene contacto o depende solo de HR?
[ ] ¿Hay bates detrás para runs?
[ ] ¿Hay tráfico delante para RBI?
[ ] ¿El precio paga la varianza?

14.1 Hits

[ ] Contacto alto.
[ ] Bajo K.
[ ] Buen matchup de mano.
[ ] Pitcher permite hits/contacto.
[ ] No depende de extra base.

14.2 Total bases

[ ] Contacto + slug.
[ ] Puede cobrar con doble o múltiples hits.
[ ] Pitcher permite hard contact.
[ ] Parque/clima ayuda.
[ ] Bateador tiene forma de elevar pelota o castigar gaps.

14.3 HR

[ ] Barrel% real.
[ ] Pull-air profile favorable.
[ ] Pitcher permite flyballs/HR.
[ ] Parque/clima ayuda.
[ ] Matchup de repertorio favorable.
[ ] Cuota paga varianza.
[ ] Es sprinkle, no stake fuerte.

14.4 Runs

[ ] Batea 1º-2º.
[ ] Se embasa bien.
[ ] Tiene velocidad.
[ ] Buenos bates detrás.
[ ] Equipo tiene TT alto.
[ ] Pitcher rival permite tráfico.

14.5 RBI

[ ] Batea 3º-5º.
[ ] Hay OBP delante.
[ ] Pitcher permite tráfico.
[ ] Bateador tiene contacto fuerte.
[ ] Equipo proyecta carreras.

14.6 H+R+RBI

[ ] Bateador está 2º-5º.
[ ] Equipo tiene TT alto.
[ ] Puede participar de varias formas.
[ ] Mejor que hit simple si el guion es ofensivo colectivo.


15. PRECIO Y LÍNEA

[ ] ¿Qué probabilidad real le doy?
[ ] ¿Qué probabilidad exige la cuota?
[ ] ¿La cuota ya cobró el edge?
[ ] ¿Estoy pagando nombre/equipo grande?
[ ] ¿La línea alternativa protege mejor?
[ ] ¿La pick sirve como single o solo parlay?
[ ] ¿Hay mejor mercado para la misma lectura?
[ ] ¿Estoy tomando una línea alta solo porque "debería pasar"?
[ ] ¿La varianza del mercado justifica stake bajo?


16. SEÑALES DE PASS

[ ] No puedo nombrar el edge principal.
[ ] El edge y el mercado elegido no coinciden.
[ ] Necesito demasiadas condiciones perfectas.
[ ] El favorito gana probable, pero la cuota está exprimida.
[ ] Me gusta el equipo, pero no el mercado.
[ ] El abridor y el bullpen dicen cosas opuestas.
[ ] El lineup y el parque dicen cosas opuestas.
[ ] El total depende de un solo equipo.
[ ] El F5 depende de timing demasiado preciso.
[ ] El TT está demasiado alto.
[ ] La prop depende de HR o clutch sin precio suficiente.
[ ] Hay mejores opciones live que pre-partido.


17. CHECKLIST FINAL RÁPIDA ANTES DE APOSTAR

[ ] ¿Cuál es el edge principal en una frase?
[ ] ¿Ese edge es temprano o full game?
[ ] ¿El mercado elegido captura ese edge directamente?
[ ] ¿Estoy agregando dependencias innecesarias?
[ ] ¿El lineup confirmado respalda la lectura?
[ ] ¿El abridor respalda la lectura?
[ ] ¿El bullpen respalda la lectura?
[ ] ¿El parque/clima no contradice la lectura?
[ ] ¿El rival tiene ruta clara para romperla?
[ ] ¿Estoy confundiendo tráfico con carreras?
[ ] ¿Estoy confundiendo pitcher vulnerable con under Ks/outs?
[ ] ¿Estoy confundiendo favorito probable con ML caro/run line?
[ ] ¿La cuota paga el riesgo?
[ ] ¿Es el mejor mercado disponible?
[ ] ¿Stake proporcional a confianza?
[ ] Decisión final: apostar / bajar stake / buscar mejor mercado / pass.
